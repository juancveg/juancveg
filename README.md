## 🚀 Featured Projects

### [BuildSafe](https://github.com/jhanhrndz/buildsafe-frontend)
Web platform for real-time PPE compliance monitoring on construction sites, combining React, Node.js/Express, and a Python/Flask + YOLOv11 AI service for automatic image-based detection.
`React` `Node.js` `Python` `YOLOv11` `MySQL` `Cloudinary`

### [TiendaYa](https://github.com/jhanhrndz/tiendaYa-project)
E-commerce platform for small local businesses in Barranquilla to go digital, built with React, Node.js, Express, and MySQL.
`React` `Node.js` `Express` `MySQL` `REST API`

### [TransporMap](https://github.com/jhanhrndz/transpormap)
Java application that lets transport operators report road hazards and obstructions in real time using geolocation and an Oracle database.
`Java` `Oracle` `Geolocation`

### [Network Analyzer Dashboard](https://github.com/juancveg/Network-Analyzer-Dashboard)
Interactive web tool that scans a local network, visualizes its topology, calculates subnets (VLSM), and identifies connected devices — based on Cisco course concepts.
`Python` `Networking` `VLSM`
